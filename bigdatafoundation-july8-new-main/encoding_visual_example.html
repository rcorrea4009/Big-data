üéØ Step 3: One-Hot Encoding (Avoid Number Bias)

Problem: ML thinks manager(2) > student(1) > teacher(0) mathematically!

Solution: Create separate binary columns for each category:

Job One-Hot Vectors:

  • teacher (index 0) ‚Üí [1, 0, 0]
  • student (index 1) ‚Üí [0, 1, 0]
  • manager (index 2) ‚Üí [0, 0, 1]

Marital One-Hot Vectors:

  • single (index 0) ‚Üí [1, 0, 0]
  • married (index 1) ‚Üí [0, 1, 0]
  • divorced (index 2) ‚Üí [0, 0, 1]

üîß Step 4: Vector Assembler (Combine Everything)

üîÑ Pipeline Magic

Instead of doing these steps manually every time, Pipeline does it all automatically:

# One pipeline handles everything!
pipeline = Pipeline(stages=[
    StringIndexer(job ‚Üí jobIndex),
    OneHotEncoder(jobIndex ‚Üí jobVec),
    StringIndexer(marital ‚Üí maritalIndex),
    OneHotEncoder(maritalIndex ‚Üí maritalVec),
    VectorAssembler([age, jobVec, maritalVec] ‚Üí features)
])

# Apply to any new data with one command!
clean_data = pipeline.fit(train).transform(new_data)
